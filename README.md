# BIGCAM
Student project : designing, build and program an autonomous plane in order to map the ground with a camera.

## Dependencies
+    CMake
+    GCC for C++11
+    OpenCV
+    GNU/Linux (not tested on Windows and MAC OS)

## To compile and execute
    cd BIGCAM
    mkdir build;
    cd build
    mkdir test
    cmake ..
    make
    ./drone
